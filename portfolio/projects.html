Book Inventory Database

In collaboration with a team of three, we developed a comprehensive web application for my Database Management Systems class. Our project involved designing and implementing a relational database, defining tables, relationships, and constraints. Using HTML and PHP, we created a user-friendly frontend that enables secure user authentication, personalized reading lists, group collaborations for managing reading materials, and a robust book rating and review system. This project showcases my proficiency in database design, SQL, frontend development, and collaborative problem-solving skills.

Plagerism Checker

In spring 2022 I took a data structures and abstraction course; in this course we were assigned a group project written in C++. My group chose to make a program that was a plagirism checker. In this project we used two hash string search algorithms, Robin-Karp and Boyer-Moore. In our project we also compared the time efficency of both algithms.
